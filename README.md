# Employee Salary Hike & Audit System (Oracle PL/SQL)

## Overview
This project demonstrates a beginner-friendly salary management system built using Oracle SQL and PL/SQL.
It focuses on applying salary hikes and automatically maintaining audit records.

## Features
- PL/SQL procedure to apply salary hikes
- Trigger to automatically log salary changes
- Audit table to store historical salary data
- Business rule enforcement at database level

## Technologies Used
- Oracle Database 21c XE
- SQL Developer
- SQL & PL/SQL

## Purpose
This project was created to practice real-world PL/SQL concepts and prepare for entry-level database and software engineering roles.
